VISUAL STUDIO CODE

Visual studio(Vs code) is a lightht weight but powerfull source code editor developed by Microsoft. it supports a wide range of programming languages and comes with features

Features of VScode:

  • Syntax highlighting and Auto-completion for most languages.
  • Integrated Terminal for running scripts or commands
  • Version control Integration for Git and other commands for running scriots or commands
  • Extentions Marketplaceto add functionality such as language-specific support,linters,debuggers,themes and more
  • Customizable Interfacewith user-defined settings and shortcuts
  • Built-in Debuggingfor a seamless coding experience.
